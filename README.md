## 📜 חוקי המערכת (חובה לקרוא)
1. **חיבור ישיר:** האפליקציה עובדת בתוך הרשת הביתית לביצועים מקסימליים.
2. **אבטחת IP:** הגישה פתוחה רק למכשיר המורשה.
3. **מצב מצלמה:** ניתן להשאיר את המצלמה דולקת שבוע שלם ללא הגבלה.
4. **עכבר חכם:** 
   - לחיצה קצרה = לחיצה שמאלית.
   - לחיצה ארוכה (600ms) = לחיצה ימנית.
