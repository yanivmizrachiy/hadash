# 🔱 TITAN OMNI-CORE V12.0 QUANTUM
### מרכז בקרה, מעקב ושליטה עולמי | יניב מזרחי - Samsung S24 Ultra

מערכת TITAN היא פלטפורמת שליטה "אפס-דמו" המאפשרת ניהול מוחלט של מחשב הסלון ומחשב החדר מכל מקום בעולם. המערכת מותאמת אישית לניצול מסך ה-120Hz ואיכות ה-Ultra HD של סמסונג גלאקסי S24 אולטרה.

---

## 🛠️ יכולות מנוע V12.0 (ביצועים פיזיים בלבד)
- **Quantum Stream:** הזרמת מסך ב-10 FPS באיכות Ultra-HD חדה.
- **Surveillance Pro:** גישה ישירה למצלמות המחשבים (Webcam) למעקב 24/7.
- **Smart Mouse:** 
  - נגיעה = קליק שמאלי.
  - נגיעה ארוכה (0.5 שנ') = **קליק ימני**.
- **Desktop Master:** כפתורי פקודה למיזעור חלונות (Win+D), מעבר בין תוכנות (Win+Tab) וסגירת חלונות (Alt+F4).
- **Power Control:** נעילה, שינה, איתחול וכיבוי פיזי של המחשבים.
- **ניטור חי:** תצוגת מדדי מעבד (CPU) וזיכרון (RAM) של כל מחשב בזמן אמת.

## 📱 התקנה והפעלה בנייד
1. **כניסה למערכת:** פתח את הקישור: [https://yanivmizrachiy.github.io/hadash/](https://yanivmizrachiy.github.io/hadash/)
2. **התקנת אייקון:** לחץ על כפתור הזהב **"התקן אפליקציה"** (או דרך תפריט הדפדפן: "התקן").
3. **חיבור עולמי:** וודא ש-**Tailscale** דולק בטלפון ובמחשב.
4. **גשר אבטחה:** בכל פעם שאתה מחליף מחשב (סלון/חדר), לחץ על הכפתור הכתום **"אישור חיבור"** פעם אחת לשבירת חסימות כרום.

---

## 📜 חוקי עבודה וסנכרון
1. **זיהוי אוטומטי:** המחשבים מעדכנים את ה-IP שלהם לקבצי `salon_status.json` ו-`room_status.json`. אין צורך בהגדרה ידנית.
2. **מצב שירות:** שרת ה-Python (`monster_server.py`) רץ במצב **Hidden** (נסתר) ואינו מפריע לעבודה השוטפת במחשב.
3. **אבטחה:** הגישה מותרת אך ורק למכשיר הסמסונג המורשה בתוך רשת ה-Mesh הפרטית.

---

## 📂 מבנה קבצי המערכת
- `index.html`: ממשק השליטה היוקרתי (Glassmorphism UI).
- `monster_server.py`: המוח שמבצע את הפקודות ומזרים וידאו.
- `manifest.json`: הגדרות האייקון והאפליקציה למסך הבית.
- `*_status.json`: נתוני ה-IP והבריאות של המחשבים.
